Gridy Example

Basic Grid

Add gy="g" to the parent and gy="c" to the children, and the columns will get automagically equal width size.

Nesting

Another feature of Gridy is the possibility of nesting directly inside the columns (again, without adding rows!).

No-Gutter Grid

Remove the gutter from the grid by adding g-no-gutter.

Horizontal Alignement

Apply horizontal alignment by adding g-left, g-center, g-right.

Horizontal distribution

Apply horizontal distribution by adding g-space-around or g-space-between.

Vertical alignment, grid level

Apply vertical alignment by adding g-top, g-middle, g-bottom.

Vertical alignment, column level

Apply vertical alignment by adding c-top, c-middle, c-bottom.

Flow and ordering

In the example below, the first grid has normal flow, the second grid is reversed g-reverse, the third grid has first and last columns inverted c-last c-first, the forth has very specific ordering order-n*.

Also, note that in the ordering you can use responsive breakpoints: order-2 order-s-1 order-m-3.

Equal height

Apply equal height by adding g-equal-height.

Column direction

Apply column direction with g-column. Reverse the column with g-column-reverse

Offset

Apply column offset with off-n*. You can also use responsive breakpoints, for example off-6 off-s-10.
